# 📸 Fotogram – Onepager Photo Portfolio

A fully handcoded, fully responsive one-page photo portfolio, built with pure JavaScript, HTML5 & CSS3 — no frameworks, no shortcuts.

---

## 💡 The Idea

Fotogram was created as a course project during my studies at **[Developer Academy](https://developer-akademie.de)** — a mandatory exercise and portfolio piece. The goal: a clean, distraction-free photo showcase that puts the work front and center — without a CMS, without bloat.

- Course project built as part of the Developer Academy curriculum
- Design provided by Developer Academy — implementation fully handcoded
- Fully handcoded in HTML5, CSS3 & Vanilla JS
- No frameworks, no templates — everything written by hand
- Focus on visual storytelling, layout clarity, and performance
- Fully responsive — optimized for all screen sizes

---

## 🎨 My Approach

Letting the photos speak, with just enough code to get out of the way:

- **Step 1: Concept & Layout** – Define visual hierarchy, grid structure, and flow
- **Step 2: HTML & CSS Foundation** – Semantic markup, minimal styling, clean typography
- **Step 3: JavaScript Layer** – Dynamic interactions, lightbox, smooth scroll — all vanilla
- **Step 4: Responsiveness** – Mobile-first approach using Flexbox/Grid and media queries
- **Step 5: Polish & Test** – Cross-device testing, performance check, visual consistency

---

## ⚡ Key Highlights

- Single-page layout — lightweight, fast, focused
- Vanilla JS only — no jQuery, no React, no excuses
- Dynamic photo grid with responsive reflow
- Lightbox / fullscreen viewer (custom-built)
- Smooth scroll navigation
- Fully responsive design — from mobile to widescreen
